# SnovyMafia

**SnovyMafia** — это полноценный плагин для **Minecraft (Paper/Purpur 1.21.8)**, реализующий игру **"Мафия"** с расширенными возможностями, настраиваемыми ролями, GUI-меню и системой ведущих.

---

## ✅ Возможности

- 🎭 **20 ролей** с уникальными способностями (мафия, доктор, шериф, маньяк, стриптизёрша и др.)
- 🎮 **Система ведущих** — управление игрой через GUI
- 📜 **Книги с ролями** и инструкциями для игроков
- ⏱ **Настройка времени** (ночь, голосование, отсчёт перед стартом)
- 📊 **Scoreboard** с отображением фазы и времени
- 🎯 **Голосование через чат** (`/vote <ник>`)
- 🧠 **Клики по игрокам** для выполнения ночных действий
- 🌙 **Синхронизация времени Minecraft** с фазами игры
- 👻 **Невидимость ночных ролей** и **тихие действия**
- 🔄 **Очередь участников** (`/mafia join`, `/mafia leave`)
- ⚙️ **GUI настройки** ролей и времени
- 📦 **Компиляция через Gradle (Groovy DSL)**

---

## 🛠 Установка

1. Соберите плагин:
   ```bash
   ./gradlew clean build

2. Скопируйте build/libs/SnovyMafia-1.0.jar в папку plugins вашего сервера.
3. Перезапустите сервер.
4. Настройте config.yml под свои нужды.

📜 Команды
/mafia Открыть GUI управления (только для ведущих)
/mafia join Записаться в игру
/mafia leave Выйти из очереди
/mafia leading add <ник>Добавить ведущего
/mafia leading remove <ник> Убрать ведущего
/vote <ник> Проголосовать за игрока (днём)

⚙️ Настройки
Все настройки находятся в config.yml:
1. night-duration: 120              # Время ночи в секундах
2. day-vote-duration: 60            # Время голосования в секундах
3. start-countdown-duration: 60     # Время отсчёта перед стартом
4. leaders: []                      # Список ведущих
5. enabled-roles:                   # Включённые роли
6. villager: true
7. mafia: true
8. don: false
9. sheriff: true
10. doctor: true
11. # ... и т.д.

🚀 Запуск игры
Станьте ведущим: /mafia leading add <ваш_ник>
Откройте GUI: /mafia
Нажмите "▶ Начать игру (60с)" или "⚡ Начать игру (сразу)"
Игроки могут записаться: /mafia join
